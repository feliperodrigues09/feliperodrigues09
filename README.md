### 🎯 O que é um Business Scientist?

> Não é o Data Scientist que domina modelos mas não entende o negócio.  
> Não é o analista de negócios que depende de outros para extrair dados.  
> É quem **fala as duas línguas**: formula a pergunta certa de negócio, coleta e modela os dados, e entrega o insight acionável.
